المحاضرة 1: مقدمة في ML
Introduction to Machine Learning
  • ML: مجال يجعل الحاسوب يتعلم بدون برمجة صريحة (Arthur Samuel 1959)
  • Supervised: بيانات مدخلة مع إجابات صحيحة — Regression & Classification
  • Unsupervised: بيانات بدون تسميات — Clustering
  • Regression: تنبؤ بقيم مستمرة (مثل: أسعار المنازل)
  • Classification: تنبؤ بفئات (0 أو 1)
أنواع البيانات: Nominal, Binary, Ordinal, Interval, Ratio
أنواع المسافات: Manhattan (L1), Euclidean (L2), Supremum (L∞)
Euclidean: d = √(Σ(xi - yi)²) Manhattan: d = Σ|xi - yi|
  • Normalization: Min-Max, Z-score, Decimal Scaling
  • Binning: Equal-Width و Equal-Depth لتنعيم البيانات
  • Data Preprocessing: Cleaning, Integration, Reduction, Transformation
Min-Max: v' = (v-min)/(max-min) × (new_max-new_min) + new_min Z-score: v' = (v - μ) / σ
📈
المحاضرة 2: Linear Regression
الانحدار الخطي بمتغير واحد ومتعدد
  • Hypothesis: h(x) = θ₀ + θ₁x (one variable)
  • θ₀ = Y-intercept (نقطة التقاطع), θ₁ = Slope (الميل)
  • Cost Function J(θ) = (1/2m) × Σ(h(xᵢ) - yᵢ)²
Gradient Descent (θ₀): θ₀ = θ₀ - α × (1/m) × Σ(h(xᵢ) - yᵢ) Gradient Descent (θ₁): θ₁ = θ₁ - α × (1/m) × Σ(h(xᵢ) - yᵢ) × xᵢ
  • α = Learning Rate (معدل التعلم) — يجب اختياره بعناية
  • Multiple Variables: h(x) = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ
  • Feature Scaling: تقسيم قيمة المتغير على أقصى قيمة له
  • Polynomial Regression: استخدام درجات أعلى للـ features
المثال: x=[2,3,4], y=[3,4,5], θ₀=2, θ₁=2, α=0.1
🔀
المحاضرة 3: Logistic Regression
الانحدار اللوجستي والتصنيف
  • يُستخدم للتصنيف (y = 0 أو 1) — ليس للانحدار
  • Sigmoid Function: g(z) = 1 / (1 + e⁻ᶻ)
  • h(x) = g(θ₀ + θ₁x) = 1 / (1 + e^(-(θ₀+θ₁x)))
Decision Boundary: h(x) = 0.5 → z = 0 If h(x) ≥ 0.5 → predict y = 1 If h(x) < 0.5 → predict y = 0
  • Cost Function (Convex): -y×log(h) - (1-y)×log(1-h)
  • Gradient Descent: نفس صيغة Linear Regression!
  • Decision Boundary مثال: θ₀=-3, θ₁=1, θ₂=1 → x₁+x₂=3
مثال: x=[3,5,10,12], y=[1,1,0,0], θ₀=2, θ₁=1
⚖️
المحاضرة 4: Overfitting & Regularization
الإفراط في التعلم والتنظيم
  • Underfitting (High Bias): النموذج لا يتعلم جيداً من التدريب
  • Overfitting (High Variance): النموذج يحفظ التدريب لكن يفشل في الاختبار
  • حلول Overfitting:
  • ① جمع بيانات تدريب أكثر
  • ② Feature Selection (اختيار المتغيرات)
  • ③ Regularization
  • ④ Cross Validation
Ridge (L2): J = J_original + λ × Σθⱼ² Lasso (L1): J = J_original + λ × Σ|θⱼ|
  • K-Fold Cross Validation: 5 ≤ k ≤ 10
  • Ridge تُقلل الـ coefficients ولكن لا تصفّرها
  • Lasso يمكنها تصفير بعض المعاملات (Feature Selection)
🔍
المحاضرة 5: KNN & Naive Bayes
K-Nearest Neighbor والبايز الساذج
  • KNN يُستخدم في Classification و Regression
  • خطوات KNN: ① حساب المسافة ② الترتيب ③ اختيار K أقرب
Euclidean Distance: d = √((x₂-x₁)² + (y₂-y₁)²) KNN Regression: Average of k nearest weights
  • للتصنيف: Majority vote بين K جيران
  • للانحدار: متوسط قيم K الأقرب
  • Naive Bayes: يعتمد على نظرية بايز للاحتمالات الشرطية
  • P(class|features) ∝ P(features|class) × P(class)
  • مثال: التنبؤ بلعب التنس بناءً على الطقس
مثال KNN Regression: k=3, query=(2,1) → avg top3 = 2
⚡
المحاضرة 6: Support Vector Machines
آلة المتجهات الداعمة
  • SVM تجد أفضل hyperplane يُعظّم الهامش بين الفئات
  • Support Vectors: النقاط الأقرب إلى الـ hyperplane
Hyperplane: b₀ + b₁x₁ + b₂x₂ = 0 (wᵀX = 0) Positive: wᵀX > 0 Negative: wᵀX < 0
  • Hard Margin (Maximal Margin Classifier): بيانات مفصولة تماماً
  • Soft Margin (Support Vector Classifier): يسمح ببعض الأخطاء
  • Kernel Trick: تحويل البيانات لأبعاد أعلى لجعلها قابلة للفصل
  • أنواع Kernel: Linear, Polynomial, RBF (Gaussian)
  • C parameter: يتحكم في عدد الأخطاء المسموح بها
كبر C → أقل أخطاء لكن خطر Overfitting | صغر C → Soft Margin
🌲
المحاضرة 7: Ensemble Methods
أساليب التجميع
  • تجميع نماذج متعددة للحصول على تنبؤ أفضل
  • Homogeneous: نفس الخوارزمية + بيانات مختلفة
  • Heterogeneous: خوارزميات مختلفة
1. Max Voting (Majority) 2. Averaging 3. Weighted Averaging 4. Bagging (Bootstrap Aggregating) → يقلل Variance 5. Boosting → يقلل Bias 6. Stacking
  • Bagging: يعيد أخذ عينات عشوائية مع الاستبدال → Random Forest
  • Boosting: يُثقّل النقاط الخاطئة → AdaBoost, XGBoost
  • Stacking: يستخدم نموذج Meta-learner للتجميع
Weighted Avg مثال: [5×0.23 + 4×0.23 + 5×0.18 + 4×0.18 + 4×0.18] = 4.41
📊
المحاضرة 8: Model Evaluation
تقييم النماذج
  • Regression Metrics: MSE, RMSE, MAE, R²
  • Confusion Matrix للتصنيف الثنائي:
  • TP: صحيح موجب | TN: صحيح سالب | FP: خاطئ موجب | FN: خاطئ سالب
Accuracy = (TP + TN) / (TP + TN + FP + FN) Precision = TP / (TP + FP) Recall = TP / (TP + FN) F1-Score = 2 × (Precision × Recall) / (Precision + Recall)
  • مثال: TP=120, FP=20, FN=10, TN=50 (إجمالي=200)
  • Accuracy = 170/200 = 0.85
  • Precision = 120/140 ≈ 0.857
  • Recall = 120/130 ≈ 0.923
F1-Score يوازن بين Precision و Recall — مفيد عند عدم توازن الفئات
